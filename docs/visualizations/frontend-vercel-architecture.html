YudaiV3 Frontend Architecture

Vite React app in src deployed on Vercel with serverless API bridge routes.

Frontend Package

  • • Entry: src/main.tsx and src/App.tsx
  • • UI: AgentWorkbench behind ProtectedRoute
  • • State: Zustand stores plus React Query hooks
  • • Build: npm run build inside src

Vercel Edge Surface

  • • src/vercel.json rewrites app paths to index.html
  • • /api/proxy forwards daifu, github, and controller APIs
  • • /api/realtime bridges backend WebSocket events to SSE
  • • /api/ai streams model responses through AI SDK

Backend Touchpoints

  • • Default backend origin is https://api.yudai.app
  • • Auth validates bearer tokens through /auth/api/user
  • • Realtime uses /controller/sessions/:id/ws/unified
  • • AI route loads and persists session context through /daifu
