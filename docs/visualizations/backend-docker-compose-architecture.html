YudaiV3 Backend Architecture

Docker Compose backend deployment with FastAPI controller, PostgreSQL, Modal sandbox provisioning, and persistent volumes.

Controller API

  • • Docker image builds from backend/Dockerfile
  • • start.sh initializes tables before server startup
  • • FastAPI mounts /auth, /github, /daifu, and /controller
  • • Healthcheck calls http://localhost:8000/health

Data Layer

  • • db service runs postgres:15 on the compose bridge
  • • init.sql is mounted into docker-entrypoint-initdb.d
  • • app writes logs, run data, artifacts, and model caches to named volumes
  • • backend waits for db health before serving traffic

Realtime Execution

  • • modal-preflight validates sandbox provisioning before backend starts
  • • Controller provisions Modal sandboxes with an encrypted :8100 tunnel
  • • Sandbox server exposes internal exec WebSocket routes
  • • Heartbeats and unified events return through controller routes
